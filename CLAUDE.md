# One Life — Project Context for Claude Code ## What this app is A Stoic-inspired iOS app (React Native + Expo) that shows users exactly how many days they have left to live — calculated from a scientific questionnaire — then lets them earn time back through daily healthy habits. ## Tech stack - React Native + Expo + TypeScript - expo-router for navigation - react-native-svg for the week grid - react-native-reanimated for animations - async-storage + zustand for state - expo-notifications for push alerts - react-native-view-shot + expo-sharing for the share card - RevenueCat for subscriptions - PostHog for analytics ## Key files - lib/lifeCalc.ts — science model (calculates days remaining) - lib/habits.ts — habit definitions + time values - lib/quotes.ts — 365 Stoic quotes - store/useLifeStore.ts — global state - app/index.tsx — home screen (clock + week grid) - app/quiz.tsx — onboarding questionnaire ## Core features to build 1. Life questionnaire (3 fast questions → clock reveal) 2. Ticking days counter + life week grid (SVG) 3. Daily habit tracker (habits add hours to clock) 4. Shareable life grid card (1080x1080 PNG via view-shot) 5. Push notifications (8am daily with personalised days count) 6. Pro paywall via RevenueCat ($2.99/mo, 7-day trial) ## Full product spec See one-life-product-bible.md for complete feature list, science model data, onboarding flow, and paywall strategy. ## Code style - TypeScript strict mode - Functional components only - No class components - Tailwind-style inline styles (no StyleSheet.create) - All async operations use async/await
